## 🎯 项目概述

墨韵·智库是一个面向年轻创作者、技术学习者、文学爱好者的**"技术 + 文学"双内容形态共存的社区平台**。

### 核心特性

- **双模式设计**: 技术流/文学派两种内容展示与推荐模式
- **三端协同**: 后端服务 + 用户前台(UniApp) + 后台管理(Vue3)
- **智能推荐**: 双通道召回、用户画像、场景化排序
- **商业化能力**: CPS分销、钱包打赏、广告位管理
- **成长体系**: 积分、等级、勋章系统
- **工作流引擎**: Flowable集成，支持复杂审批流程

### 技术栈总览

| 模块 | 技术栈 |
|------|--------|
| **后端服务** | Spring Boot 3.3.2 + JDK 21 + MyBatis-Plus 3.5.7 + MySQL 8.0 + Redis + Elasticsearch + Flowable 7.1.0 |
| **用户前台** | Uni-app 3.0 + Vue 3.4.21 + Pinia 2.1.7 + uView-Plus 3.1.30 + Vite 5.2.8 |
| **后台管理** | Vue 3.4.0 + Vite 5.0.4 + Element Plus 2.4.3 + Pinia 2.1.7 + bpmn-js 11.1.0 |
