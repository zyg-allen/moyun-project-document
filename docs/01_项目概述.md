## 一、项目简介

**墨韵·智库 (Moyun Wisdom)** 是一个面向年轻创作者、技术学习者、文学爱好者的**文学+技术双内容形态共存的社区平台**。项目采用现代化技术栈，构建完整的内容生产、分发推荐、用户成长、商业化体系。

### 核心特性

| 特性 | 说明 |
|------|------|
| 🎨 **品牌定位** | "一纸墨"，取自"我有一纸墨，足以慰风尘" |
| 🏢 **前后端分离** | Vue 3 + Spring Boot 3 架构 |
| 🔐 **独立认证体系** | 前后台用户完全独立，双 SecurityFilterChain 配置 |
| 🎨 **多主题支持** | 日间、夜间、护眼三种主题自由切换 |
| 📱 **响应式设计** | 完美适配各种屏幕尺寸 |
| 📊 **工作流引擎** | Flowable 集成，支持复杂的业务审批流程 |
| ✍️ **双编辑器模式** | 富文本和 Markdown 双编辑器支持 |

**品牌设计文档**: [moyun-portal/网站名称设计.md](../moyun-portal/网站名称设计.md)

**独立认证方案**: [moyun-server/独立认证方案说明.md](../moyun-server/独立认证方案说明.md)

## 三、模块划分

### 3.1 后端服务 (moyun-server)

| 模块 | 职责 | 说明 |
|------|------|------|
| **common/** | 通用基础组件 | 注解、常量、枚举、异常、返回格式 |
| **core/** | 核心框架 | 安全认证、配置、MVC扩展、切面 |
| **util/** | 工具类 | 日期、加密、文件、HTTP等工具 |
| **modules/system/** | 系统管理 | 用户、角色、菜单、权限管理 |
| **modules/portal/** | 门户业务 | 前台用户、文章、评论、互动 |
| **ext/cms/** | 内容管理 | 后台CMS管理功能 |
| **ext/job/** | 定时任务 | Quartz定时任务 |
| **ext/generator/** | 代码生成 | 代码生成器 |
| **ext/flowable/** | 工作流 | Flowable工作流引擎 |

### 3.2 前台门户 (moyun-portal)

| 模块 | 职责 | 说明 |
|------|------|------|
| **pages/** | 页面组件 | 首页、文章详情、用户中心等 |
| **components/** | 公共组件 | 导航栏、文章卡片、评论区等 |
| **api/** | API接口 | 前端API调用封装 |
| **stores/** | 状态管理 | Pinia状态管理 |
| **composables/** | 组合式函数 | 可复用的逻辑 |

### 3.3 后台管理 (moyun-admin-vue)

| 模块 | 职责 | 说明 |
|------|------|------|
| **views/cms/** | CMS页面 | 用户管理、文章管理、分类管理等 |
| **views/system/** | 系统管理 | 菜单、角色、参数配置等 |
| **views/monitor/** | 系统监控 | 日志、在线用户、服务器状态等 |
| **views/flowable/** | 工作流 | 流程设计、任务审批等 |

## 五、快速入门

### 环境要求

| 软件 | 版本 | 说明 |
|------|------|------|
| JDK | 21+ | Java开发环境 |
| Node.js | 18+ | JavaScript运行环境 |
| MySQL | 8.0+ | 关系型数据库 |
| Redis | 6.0+ | 缓存中间件 |

### 启动命令

```bash
# 进入项目目录
cd moyun-project-document

# 查看帮助
./scripts/dev.sh status

# 安装前端依赖
./scripts/dev.sh install

# 启动前台
./scripts/dev.sh portal

# 启动后台
./scripts/dev.sh admin

# 启动后端
./scripts/dev.sh server
```

### 访问地址

| 服务 | 地址 | 说明 |
|------|------|------|
| 后端API | http://localhost:8080 | Spring Boot服务 |
| 前台门户 | http://localhost:3000 | Vue前端 |
| 后台管理 | http://localhost:80 | Element Plus后台 |
| Swagger文档 | http://localhost:8080/doc.html | API文档 |
