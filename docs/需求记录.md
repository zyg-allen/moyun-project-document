## 项目总体需求

### 项目定位

墨韵·智库是一个面向年轻创作者、技术学习者、文学爱好者的**文学+技术双内容形态共存的社区平台**。

### 核心目标

- ✅ 构建完整的内容生产、分发推荐体系
- ✅ 实现用户成长与互动体系
- 🔄 开发功能完善的后台管理系统
- ⏳ 集成第三方登录（微信、支付宝）
- ⏳ 构建商业化体系（VIP、打赏等）

## 部署和使用说明

### 部署步骤

1. **执行数据库脚本（按顺序）
   ```sql
   -- 01_moyun_init.sql
   -- 02_workflow_init.sql
   -- 03_portal_init.sql
   -- 04_cms_menu_init.sql （新增）
   ```

2. **启动后端服务**
   - 确保数据库和Redis连接正常
   - 运行`MoyunApplication.java`

3. **启动前台应用**
   - 前台门户：`cd moyun-portal && npm run dev`
   - 后台管理：`cd moyun-admin-vue && npm run dev`

4. **访问地址
   - 前台门户：`http://localhost:5173`
   - 后台管理：`http://localhost:80`（默认若依管理系统地址）
   - API文档：`http://localhost:8080/doc.html`

### 功能验证

**后台管理系统使用
- 登录后台管理系统
- 检查「内容管理」菜单
- 验证门户用户、文章、分类、标签、评论管理功能
