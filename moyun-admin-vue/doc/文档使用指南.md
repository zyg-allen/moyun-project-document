## ❓ 常见问题

### Q1: 我应该先看哪个文档？

**A**: 如果你是第一次接触项目，请先阅读 **README.md**。它包含了所有必要的基础信息。

### Q2: 如何快速找到某个功能的实现位置？

**A**: 
1. 在 README.md 的"功能模块"章节找到模块名称
2. 在"项目结构"章节找到对应的目录
3. 进入该目录查看具体文件

### Q3: 工作流设计器如何使用？

**A**: 
1. 进入"工作流管理" → "流程定义"
2. 点击"设计流程"按钮
3. 使用 BPMN 设计器拖拽元素
4. 保存并部署流程

### Q4: 如何配置权限？

**A**: 
1. 在"系统管理" → "菜单管理"中配置菜单权限
2. 在"角色管理"中为角色分配菜单
3. 在"用户管理"中为用户分配角色
4. 前端使用 `v-hasPermi` 指令控制按钮权限

### Q5: 代码生成器如何使用？

**A**: 
1. 进入"系统工具" → "代码生成"
2. 导入数据库表
3. 配置生成选项
4. 预览并下载代码
5. 将生成的代码复制到项目中
