## 墨韵智库 - 数据库初始化脚本

### 执行顺序

请按以下顺序依次执行 SQL 脚本：

1. **01_moyun_init.sql** - 系统管理基础脚本
   - 包含：系统用户、角色、菜单、部门、岗位、字典、参数配置、操作日志、定时任务等基础表
   - 必须最先执行

2. **02_workflow_init.sql** - 工作流与定时任务脚本
   - 包含：Quartz 定时任务表、Flowable 工作流相关表
   - 在 01 之后执行

3. **03_portal_init.sql** - 门户系统脚本
   - 包含：门户用户、文章、分类、标签、评论、收藏、关注、通知、VIP、订单、钱包等表
   - 在 02 之后执行（可选，根据需要）

### 注意事项

- 所有脚本均针对 MySQL 数据库
- 执行前请确保数据库已创建并具有相应权限
- 建议在执行前备份现有数据
