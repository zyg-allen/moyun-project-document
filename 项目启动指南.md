# 墨韵智库项目启动指南

## 📋 项目概述

墨韵智库是一个现代化的知识分享社区平台，包含三个部分：
- **前台门户** (moyun-portal) - 用户使用的知识分享平台
- **后台管理系统** (moyun-admin-vue) - 管理员管理后台
- **后端服务** (moyun-server) - Spring Boot 后端 API

---

## 🚀 快速启动

### 环境要求

- **JDK**: 21+
- **Node.js**: 16+
- **Maven**: 3.6+
- **IDE**: IntelliJ IDEA / VS Code

---

## 1️⃣ 后端服务启动

### 配置说明

项目已配置 **H2 内存数据库**，无需额外安装数据库！

- 数据库会在应用启动时自动初始化
- 内置了测试数据（用户、文章、分类等）
- 可以通过 H2 控制台访问：`http://localhost:8080/h2-console`
  - JDBC URL: `jdbc:h2:mem:moyundb`
  - 用户名: `sa`
  - 密码: (空)

### 启动步骤

```bash
# 进入后端项目目录
cd moyun-server

# 编译项目
mvn clean install -DskipTests

# 启动项目
mvn spring-boot:run
```

或者在 IDE 中直接运行 `MoyunApplication.java` 主类

### 验证启动

启动成功后，访问以下地址验证：

- **后端服务**: http://localhost:8080
- **H2 数据库控制台**: http://localhost:8080/h2-console
- **API 文档 (Knife4j)**: http://localhost:8080/doc.html

## 🔧 开发建议

### 切换到 MySQL 数据库

如需使用 MySQL 数据库，修改 `application.yml`：

```yaml
spring:
  datasource:
    type: com.alibaba.druid.pool.DruidDataSource
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/moyundb?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
```

### API 接口规范

所有前台 API 路径以 `/api` 开头：
- `POST /api/user/login` - 用户登录
- `POST /api/user/register` - 用户注册
- `GET /api/article/list` - 获取文章列表
- `GET /api/article/{id}` - 获取文章详情

统一响应格式：
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 开发时注意事项

1. **端口占用**:
   - 后端: 8080
   - 前台: 3000
   - 后台: 80

2. **H2 数据库**:
   - 内存数据库，重启会清空数据
   - 如需持久化，修改 JDBC URL 为文件模式

3. **API 代理**:
   - 前台和后台都配置了代理，避免跨域问题

## 📞 常见问题

### Q: 后端启动失败怎么办？

A: 检查是否端口被占用，或修改 `application.yml` 中的端口号。

### Q: 数据库连接失败？

A: 项目已配置 H2 内存数据库，无需额外安装。如使用 MySQL，请确保服务已启动。

### Q: 前端页面空白？

A: 检查浏览器控制台是否有错误，确认后端服务是否正常启动。
