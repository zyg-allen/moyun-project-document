## 🎯 技术亮点

### 1. 统一的 API 响应格式

- `ApiResponse<T>` - 通用响应格式
  - `code`: 状态码
  - `message`: 消息
  - `data`: 数据
- `PageResult<T>` - 分页响应格式
  - `list`: 数据列表
  - `total`: 总数
  - `page`: 当前页
  - `pageSize`: 每页数量

### 2. 完整的 RESTful API 设计

- 统一的 URL 前缀 `/api`
- 标准的 HTTP 方法（GET, POST, PUT, DELETE）
- 统一的错误处理

### 3. 前后端分离架构

- 前台门户（Vue 3 + TypeScript）
- 后台管理（Vue 3 + Element Plus）
- 后端服务（Spring Boot 3 + MyBatis Plus）

### 4. 模块化设计

- 用户模块
- 内容模块（文章、分类、标签、评论）
- 成长体系模块
- 钱包模块

## 🚀 快速启动指引

### 后端服务启动

```bash
cd moyun-server
# 配置数据库连接
# 启动服务
mvn spring-boot:run
```

### 后台管理系统启动

```bash
cd moyun-admin-vue
npm install
npm run dev
```

### 前台门户启动

```bash
cd moyun-portal
npm install
npm run dev
```
