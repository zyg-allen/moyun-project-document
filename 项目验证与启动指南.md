# 墨韵智库项目验证与启动指南

## 📋 项目概览

本项目包含三个主要部分：
- **前台门户** (moyun-portal)：Vue 3 + TypeScript 构建的用户界面
- **后台管理系统** (moyun-admin-vue)：Vue 3 + Element Plus 构建的管理界面  
- **后端服务** (moyun-server)：Spring Boot 3 后端 API

## 🚀 项目启动指南

### 第一步：启动后端服务

#### 前置要求
- JDK 21+
- Maven 3.6+

#### 启动步骤
```bash
cd moyun-server

# 编译项目
mvn clean install -DskipTests

# 启动应用
mvn spring-boot:run
```

或者在 IDE 中直接运行 `MoyunApplication.java` 主类。

#### 验证后端
启动成功后，访问以下地址验证：
- 后端服务：http://localhost:8080
- H2 控制台：http://localhost:8080/h2-console
  - JDBC URL：`jdbc:h2:mem:moyundb`
  - 用户名：`sa`
  - 密码：（留空）
- API 文档：http://localhost:8080/doc.html

#### 测试账号
| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 超级管理员 |
| test | admin123 | 普通用户 |

## 📝 技术栈总结

### 后端技术
- Spring Boot 3.3.2
- Spring Security + JWT
- MyBatis Plus
- H2 Database（开发环境）/ MySQL（生产环境）
- Knife4j（API 文档）
- Druid（数据库连接池）

### 前端技术（前台门户）
- Vue 3 + TypeScript
- Vite
- Tailwind CSS
- Pinia（状态管理）

### 前端技术（后台管理系统）
- Vue 3 + Element Plus
- Vue Router
- Pinia（状态管理）
- Axios（HTTP 请求）
